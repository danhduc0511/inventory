-Tích hợp redis docker:
       +B1: Tải và cài đặt Docker
       +B2 thêm dp spring-boot-starter-data-redis vào file pom.xml
       +B3 :Tạo file docker-compose.yml đẻ cấu hình redis
             services:
               redis:
                 image: "redis:latest"
                 container_name: "redis_container"
                 ports:
                   - "6379:6379"
       +B4: Chạy lệnh docker-compose up -d để khởi động redis
       +B5:Cấu hình redis(same mysql):
           +creare class RedisConfig để cấu hình kết nối redis
           +Trong file application.properties thêm các cấu hình kết nối redis
                spring.redis.host=localhost
                spring.redis.port=6379
